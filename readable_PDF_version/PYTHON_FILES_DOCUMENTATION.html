contrarian_earnings_analyzer.py

src/analyzers/contrarian_earnings_analyzer.py
Analyzer Core Analysis

üìù Description

Identifies contrarian analysts who were minority voices before earnings but were proven right when actual earnings results were released. This is the core analysis engine of the system.

üéØ Analysis Workflow

  1. Search pre-earnings articles
  2. Analyze sentiment and predictions
  3. Compare with actual earnings
  4. Identify contrarian voices
  5. Score contrarian accuracy

üìä Data Sources

  • Guardian API for articles
  • Groq API for sentiment analysis
  • Yahoo Finance for earnings data

üöÄ Key Features

  • Pre-Earnings Analysis: Collect articles before earnings date
  • Sentiment Scoring: AI-powered sentiment analysis
  • Contrarian Detection: Identify minority but correct voices
  • Performance Tracking: Historical accuracy measurement

üèóÔ∏è Main Class: ContrarianEarningsAnalyzer

  • collect_pre_earnings_articles(company, earnings_date, days_before=30)
  • analyze_article_sentiment(article_text)
  • get_earnings_prediction(article_text, company)
  • get_actual_earnings_result(symbol, earnings_date)
  • identify_contrarians(analyzed_articles, actual_result)
